Авторизация

Работа с приложением начинается с процесса авторизации, где необходимо ввести логин и пароль.

К навигации

Работа с пользователями

После успешной авторизации для администратора становится доступен просмотр списка пользователей.

Администратор может выполнять поиск конкретного пользователя по логину с помощью специального поля.

Администратор может выбирать любого пользователя из списка и переходить к просмотру его личной статистики, нажав на карточку пользователя.

Администратор может добавлять нового пользователя, нажав на кнопку

К навигации

Добавление пользователя

Администратор может добавлять нового пользователя, введя его логин и пароль.

После ввода данных для добавления пользователя в систему администратору необходимо нажать на кнопку

К навигации

Просмотр статистики пользователя

После того, как администратор нажал на карточку пользователя, ему становится доступна личная статистика пользователя.

Администратор может блокировать пользователю доступ к системе, нажав на кнопку

Администратору доступно удаление пользователя из системы по кнопке

К навигации

Работа с упражнениями

Для просмотра списка всех упражнений администратору необходимо нажать на кнопку

Администратор может фильтровать список упражнений по уровню сложности и длине упражнения.

Администратору доступно удаление упражнений из системы по кнопке

Администратор может добавлять новое упражнение, нажав на кнопку

К навигации

Добавление упражнения

Администратор может добавлять новое упражнение, введя его параметры.

При выборе ручного типа создания упражнения администратору необходимо самостоятельно ввести символьную последовательность в появившемся поле.

После ввода данных для добавления упражнения в систему администратору необходимо нажать на кнопку

К навигации

Просмотр общей статистики

Для просмотра общей статистики администратору необходимо нажать на кнопку

К навигации

Выход из приложения

Для выхода из приложения "Кейскилл" администратору необходимо нажать на кнопку

После этого появится окно с просьбой подтверждения выхода из приложения.

В случае подтверждения выхода администратор попадет на страницу авторизации.
