Авторизация

Работа с приложением начинается с процесса авторизации, где необходимо ввести логин и пароль.

К навигации

Взаимодействие со списком упражнений

После успешной авторизации для обучающегося становится доступен просмотр списка упражнений.

Обучающийся может фильтровать список упражнений по уровню сложности и длине упражнения.

Обучающийся может выбирать любое упражнение из списка и переходить к его выполнению, нажав на карточку упражнения.

Выполнение упражнения

На странице выполнения упражнения обучающийся может вводить символьную последовательность, выводящуюся на экране и просматривать текущую статистику.

другой скрин

Обучающемуся также доступно отображение или скрытие виртуальной клавиатуры с помощью чекбокса.

Взаимодействие со списком пользователей

Для просмотра списка всех пользователей обучающемуся необходимо нажать на кнопку

Обучающийся может выполнять поиск конкретного пользователя по логину с помощью специального поля.

Выход из приложения

Для выхода из приложения "Кейскилл" обучающемуся необходимо нажать на кнопку

После этого появится окно с просьбой подтверждения выхода из приложения.

В случае подтверждения выхода обучающийся попадет на страницу авторизации.
